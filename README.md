Objetivo AgroGuardian: 
Identificar os problemas presentes no solo para saber o motivo de não estar gerando uma boa safra.

Justificativa:
Meu projeto do agrinho se relaciona com o tema já que a finalidade dele é unir tecnologia e sustentabilidade para ajudar os agricultores, tanto pequenos, quanto grandes, para acharem a melhor forma de seu solo ser produtivo,
mas ao mesmo tempo respeitarem o meio ambiente para que tenha não tanta poluição. Ele além de mostrar os problemas no solo, tem uma página especifica para ajudar micro agricultores a terem uma boa plantação sabendo uma média de quanto eles iriam gastar para solucionar os problemas presentes no solo.
